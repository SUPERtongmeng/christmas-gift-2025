# Christmas Tree 3D Web Project

This is a 3D interactive Christmas Tree web application.

## How to deploy to GitHub Pages

1.  **Create a Repository**: Go to [GitHub](https://github.com) and create a new repository (e.g., `christmas-tree-2025`).
2.  **Upload Files**: Upload all the files in this folder (`index.html` and all images) to your new repository.
3.  **Settings**:
    *   Go to the repository **Settings**.
    *   Click on **Pages** in the left sidebar.
    *   Under **Build and deployment** > **Branch**, select `main` (or `master`) and `/ (root)`.
    *   Click **Save**.
4.  **Visit**: Wait a minute, and GitHub will give you a link (e.g., `https://your-username.github.io/christmas-tree-2025/`). Click it to see your live site!
